### `// О ПРОЕКТЕ`

</div>

<br>

**UNTITLED** — это десктопное приложение на базе **Electron**, которое превращает SoundCloud в нативный музыкальный плеер. Кастомный интерфейс, нулевая реклама, Discord Rich Presence и полная библиотека SoundCloud — всё в одном окне.

Создано потому что браузерный SoundCloud — это боль.

### `// УСТАНОВКА`

</div>

<br>

<details>
<summary><b>🪟 Windows</b></summary>
<br>

1. Перейди в [**Releases**](../../releases/latest)
2. Скачай файл `UNTITLED-Setup-x.x.x.exe`
3. Запусти инсталлер и следуй инструкциям
4. Открой приложение и войди через Google

<br>

> **Примечание:** Windows SmartScreen может показать предупреждение при первом запуске — нажми «Подробнее» → «Всё равно запустить».

</details>

<details>
<summary><b>🍎 macOS</b></summary>
<br>

1. Перейди в [**Releases**](../../releases/latest)
2. Скачай файл `UNTITLED-x.x.x.dmg`
3. Открой `.dmg` и перетащи UNTITLED в `Applications`
4. При первом запуске: **правой кнопкой → Открыть**

<br>

> **Важно:** приложение не имеет подписи Apple. Это нормально для open-source проекта — используй контекстное меню для первого запуска.

</details>

### `// РАЗРАБОТКА`

</div>

<br>

**Требования**

| Инструмент | Версия |
|---|---|
| Node.js | `18+` |
| npm | `9+` |

<br>

**Клонирование и запуск**

```bash
# Клонировать репозиторий
git clone https://github.com/your-username/untitled
cd untitled

# Установить зависимости
npm install

# Запустить в режиме разработки
npm start
```

<br>

**Сборка**

```bash
npm run build:win     # → dist/UNTITLED-Setup.exe
npm run build:mac     # → dist/UNTITLED.dmg
npm run build         # → обе платформы
```

<br>

**Структура проекта**

```
untitled/
│
├── 📄 index.html        # UI — разметка приложения
├── 📄 renderer.js       # Renderer process — вся логика
├── 📄 main.js           # Main process — Electron
├── 📄 preload.js        # Preload скрипт / IPC bridge
├── 📄 package.json
│
└── 📁 assets/
    ├── icon.png
    ├── icon.ico
    └── icon.icns
```

### `// ЧАСТО ЗАДАВАЕМЫЕ ВОПРОСЫ`

</div>

<br>

<details>
<summary><b>Нужен ли платный SoundCloud аккаунт?</b></summary>
<br>
Нет. Приложение работает с обычным бесплатным аккаунтом SoundCloud.
</details>

<details>
<summary><b>Почему нужен Google-аккаунт?</b></summary>
<br>
Google OAuth используется как основной способ идентификации пользователя. Это нужно для сохранения настроек и привязки SoundCloud-сессии.
</details>

<details>
<summary><b>Безопасно ли это?</b></summary>
<br>
Да. Приложение не хранит пароли. SoundCloud-сессия работает через веб-авторизацию, токены хранятся локально на твоём устройстве.
</details>

<details>
<summary><b>Выбивает SoundCloud-сессию?</b></summary>
<br>
Это известная особенность — SoundCloud периодически сбрасывает веб-сессии. Используй кнопку «↺ Переподключить сессию SC» в профиле или настройках.
</details>
